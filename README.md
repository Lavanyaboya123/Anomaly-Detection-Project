# 🌫️ Advanced Anomaly Detection in Time Series

This project detects unusual patterns (anomalies) in air quality data.

## 🔍 Techniques Used
- Z-Score (Statistical)
- Isolation Forest (Machine Learning)
- KNN - PyOD (Advanced ML)

## 📊 Features
- Interactive Streamlit Dashboard
- AQI Trend Visualization
- Anomaly Highlighting

## 🚀 Run Locally
```bash
streamlit run app.py
